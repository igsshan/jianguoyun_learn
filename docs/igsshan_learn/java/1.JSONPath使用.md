# JSONPath的基础使用

> JSONPath可以快速提取json固定位置,范围的数据,而不需要创建单独的JSONObject对象.
>
> 但是如果每次处理的json格式会改变,则该方式不太适用

## 支持的语法

| -------- | --------------------------------- |
| $        | 跟对象,例如 $.name                |
| [num]    | 数组访问,其中num是数字,可以是负数 |
